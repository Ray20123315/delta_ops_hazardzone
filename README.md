# Delta Ops: Hazard Zone

這是一個基於 Forge 1.20.1 的 Minecraft 模組，目前專注於「撤離、經濟、死亡懲罰與安全箱」這幾個核心玩法。

## 目前已實裝的功能

### 1. 撤離系統
- 玩家可進入撤離區域後開始撤離流程
- 撤離完成後會結算獎勵
- 撤離成功後會將玩家傳送回大廳
- 支援記錄與查找地圖對應的撤離點

### 2. 經濟系統
- 內建哈夫幣餘額系統
- 撤離成功時會根據地圖與物品內容結算收益
- 支援物品價格設定與價格重載
- 物價資料可由管理員在遊戲內更新

### 3. 安全箱系統
- 提供安全箱能力，讓玩家可將高價值物品存入其中
- 死亡後會保留安全箱內容，避免重要物資直接遺失
- 與死亡懲罰流程整合，讓安全箱內物品不會被一併丟棄

### 4. 死亡懲罰系統
- 玩家死亡時會觸發掉落懲罰
- 普通物品與裝備可能掉落在死亡位置
- 安全箱內的物品會被保留下來

### 5. 管理員設定介面
- 已實作管理員設定 GUI
- 可選擇地圖並設定該地圖的撤離點
- 可重載物價表
- 可透過 GUI 修改物品價格
- 介面中已明確標示以下項目目前暫不支援：
  - 直接選擇安全箱等級
  - 調整死亡掉落規則
  - 設定撤離倒數時間與地圖獎勵倍率

### 6. 指令系統
目前已提供以下核心指令：

- /dt extract add <mapName>
  - 為指定地圖設定撤離點
- /dt balance
  - 查詢目前哈夫幣餘額

## 開發與建置
- Minecraft 版本：1.20.1
- Forge 版本：1.20.1
- Java 版本：17
- 建置工具：Gradle

### 建置方式
在專案根目錄執行：

```bash
./gradlew build
```

Windows 可使用：

```powershell
./gradlew.bat build
```

## 目前狀態
這個模組目前已具備「撤離 + 經濟 + 安全箱 + 管理員設定」的核心流程，並且已成功通過編譯驗證。
